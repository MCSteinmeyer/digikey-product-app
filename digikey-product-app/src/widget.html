DigiKey BOM Import

Paste a BOM. Get stock and price. Export it to Drive.

Drop in the tab-delimited BOM you shared, choose the input and output filenames, and the app will look up each line item against DigiKey. If Google Drive upload is configured, the result is written directly to your Drive.

Import and lookup

The importer expects the first row to be a header row with tab-separated columns like Item, Reference Designators, Qty, Manufacturer, Manufacturer Part Number, Description, and Package.
